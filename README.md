# WPF Image Editor

A lightweight desktop image editing tool built with C# and WPF.
This project focuses on building a simple image editor with core painting and editing features.

---

## Features (MVP)

- Create new images  
- Open and save PNG/JPG files  
- Brush and eraser tools  
- Color picker and brush size control  
- Undo / Redo  
- Zoom and pan
